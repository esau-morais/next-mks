## 🛠️ Setup

1. Clone

```
gh repo clone esau-morais/next-mks
```

2. Installing (using `pnpm`)

```
pnpm install 
```

3. Running (using `pnpm`)

```
pnpm dev
```

4. Extra: Testing (using `pnpm`)

```
pnpm test
```
